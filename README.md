# Netflix-Power BI Dashboard

## Overview
A Netflix Power BI dashboard that lets users browse and search for movies and TV shows.

## Features
- Search functionality for movies and TV shows.
- Sorting by Cast, rating, and Top 10 Genres
- Dark mode toggle.
